# Plug SiteGuard UI

**현대적이고 접근성 높은 UI 컴포넌트 라이브러리**

React 19, TypeScript 5.8, Tailwind CSS v4로 구축된 전문 UI 라이브러리입니다.

## 디자인 시스템

이 라이브러리는 **Atomic Design** 원칙을 따르며, 복잡도에 따라 컴포넌트를 구성합니다:

- **Atoms** (원자): 기본 빌딩 블록 (Button, Input, Badge 등)
- **Molecules** (분자): Atom의 간단한 조합
- **Organisms** (유기체): 복잡하고 기능이 완전한 컴포넌트
- **Templates** (템플릿): 페이지 레벨 레이아웃

## 기술 스택

- ⚛️ **React 19** - 최신 React 기능 활용
- 🔷 **TypeScript 5.8** - 완전한 타입 안정성
- 🎨 **Tailwind CSS v4** - 유틸리티 기반 스타일링
- ♿ **Radix UI** - 접근성 우선 프리미티브
- 📚 **Storybook 8.6** - 인터랙티브 문서화

### 색상 사용하기

모든 색상은 Tailwind CSS 클래스를 통해 사용할 수 있습니다:

```tsx
// Primary (Blue)
<div className="bg-primary-600 text-white">주요 버튼</div>

// Success (Green)
<div className="bg-success-600 text-white">성공 메시지</div>

// Warning (Amber)
<div className="bg-warning-600 text-white">경고 알림</div>

// Error (Red)
<div className="bg-error-600 text-white">오류 메시지</div>

// Gray (Neutral)
<div className="bg-gray-100 text-gray-900">중립 배경</div>

// 텍스트 색상
<p className="text-primary-600">주요 텍스트</p>
<p className="text-success-700">성공 텍스트</p>
<p className="text-error-600">오류 텍스트</p>

// 테두리 색상
<div className="border border-gray-300">회색 테두리</div>
<div className="border-2 border-primary-500">주요 테두리</div>
```

### 시맨틱 색상 사용법

- **Primary**: 주요 브랜드 액션, 링크, 포커스 상태
- **Success**: 확인, 성공적인 작업, 긍정적 피드백
- **Warning**: 주의 메시지, 잠재적 문제, 주의 필요
- **Error**: 파괴적 액션, 오류 상태, 중요 알림
- **Gray**: 텍스트, 배경, 테두리, 비활성 상태

## 시작하기

### 설치

```bash
pnpm add @plug-siteguard/ui
```

### 사용법

앱에 전역 스타일을 import합니다:

```tsx
import '@plug-siteguard/ui/globals.css';
```

Atomic 레벨에서 컴포넌트를 import합니다:

```tsx
// Atoms에서 import
import { Button } from '@plug-siteguard/ui/atoms';

// 또는 메인 엔트리에서 import
import { Button } from '@plug-siteguard/ui';

function App() {
  return (
    <Button variant="default" size="lg">
      클릭하세요
    </Button>
  );
}
```

### 유틸리티

조건부 클래스명을 위한 `cn` 유틸리티를 사용합니다:

```tsx
import { cn } from '@plug-siteguard/ui/utils';

const buttonClass = cn(
  'base-class',
  isActive && 'active-class',
  isDisabled && 'disabled-class'
);
```
