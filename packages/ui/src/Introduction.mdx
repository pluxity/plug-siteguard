# Plug SiteGuard UI

A modern, accessible UI component library built with React 19, TypeScript 5.8, and Tailwind CSS v4.

## Design System

This library follows **Atomic Design** principles, organizing components by complexity:

- **Atoms**: Basic building blocks (Button, Input, Badge, etc.)
- **Molecules**: Simple combinations of atoms
- **Organisms**: Complex, feature-complete components
- **Templates**: Page-level layouts

### Using Colors

All colors are available through Tailwind CSS classes:

```tsx
// Primary (Blue)
<div className="bg-primary-600 text-white">Primary button</div>

// Success (Green)
<div className="bg-success-600 text-white">Success message</div>

// Warning (Amber)
<div className="bg-warning-600 text-white">Warning alert</div>

// Error (Red)
<div className="bg-error-600 text-white">Error message</div>

// Gray (Neutral)
<div className="bg-gray-100 text-gray-900">Neutral background</div>

// Text colors
<p className="text-primary-600">Primary text</p>
<p className="text-success-700">Success text</p>
<p className="text-error-600">Error text</p>

// Border colors
<div className="border border-gray-300">Gray border</div>
<div className="border-2 border-primary-500">Primary border</div>
```

### Semantic Color Usage

- **Primary**: Main brand actions, links, focus states
- **Success**: Confirmations, successful operations, positive feedback
- **Warning**: Cautionary messages, potential issues, attention needed
- **Error**: Destructive actions, error states, critical alerts
- **Gray**: Text, backgrounds, borders, disabled states

## Getting Started

### Installation

```bash
pnpm add @plug-siteguard/ui
```

### Usage

Import the global styles in your app:

```tsx
import '@plug-siteguard/ui/globals.css';
```

Import components from atomic levels:

```tsx
// Import from atoms
import { Button } from '@plug-siteguard/ui/atoms';

// Or import from main entry
import { Button } from '@plug-siteguard/ui';

function App() {
  return (
    <Button variant="default" size="lg">
      Click me
    </Button>
  );
}
```

### Utilities

Use the `cn` utility for conditional class names:

```tsx
import { cn } from '@plug-siteguard/ui/utils';

const buttonClass = cn(
  'base-class',
  isActive && 'active-class',
  isDisabled && 'disabled-class'
);
```
